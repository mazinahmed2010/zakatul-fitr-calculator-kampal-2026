## Introduction

This project is a calculator for Zakatul Fitr designed for the Kampala region in 2026. It helps users compute their Zakatul Fitr accurately and efficiently.

## Usage

To use the Zakatul Fitr Calculator, simply input the necessary values, and the program will calculate the Zakatul Fitr owed based on the current year’s rates.

## Features

- Real-time calculations
- Easily accessible interface
- Localized currency options
